# EduMonitorAI

## Student Learning and Parent Monitoring System

EduMonitor AI is a Flask-based Student Learning and Parent Monitoring System that tracks PDF study progress, monitors application usage, generates AI-powered quizzes, and provides detailed study reports for parents and students.

EduMonitor AI is a mini-project developed using Python Flask and SQLite.

### Current Features

* Student Registration
* Parent Registration
* Student Login
* Parent Login
* Secure Password Hashing
* Parent-Student Linking using Access Code

### Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* JavaScript

### Project Objective

To develop a system that allows students to study PDF materials while enabling parents to monitor learning progress, application usage, quiz performance, and study reports.


### Status

Project is currently under development.
